# SpyGlass AI 🔍

- AI-powered competitor intelligence and startup research platform built to help founders, marketers, sales teams, product teams, and investors understand the market faster and make smarter business decisions.

- SpyGlass AI uses AI agent orchestration to crawl startup websites, discover competitors, analyze SEO and positioning, generate SWOT reports, and provide actionable business insights automatically.

# 🚀 Features

## 🌐 Website Crawling

Automatically crawls startup websites and extracts structured business data, product details, messaging, and market information.

## 🤖 AI Startup Analysis

Uses large language models to analyze startups like a business strategist, investor, and marketer.

## 🕵️ Competitor Discovery

Finds direct and indirect competitors from Product Hunt, search engines, startup directories, and public web data.

## 📈 SEO Intelligence

Generates SEO insights including keyword opportunities, metadata analysis, content positioning, and search visibility.

## 🧠 SWOT Analysis

Automatically creates detailed Strengths, Weaknesses, Opportunities, and Threats reports.

## 💬 AI Business Analyst Chat

Ask strategic questions about any analyzed company and receive contextual AI-powered insights instantly.

## 📊 Persistent Dashboard

Store and revisit analyses, compare companies, and track research over time.

## 🔐 Authentication System

Secure JWT-based authentication and protected routes.

## ⚡ AI Agent Orchestration

Multiple AI agents collaborate to perform crawling, research, competitor discovery, SEO analysis, and strategic reasoning in parallel.

## 🎯 Sales & Marketing Intelligence

Helps sales and marketing teams understand customer pain points, improve outreach strategies, analyze competitor offerings, and identify what to offer potential clients.

## 🧩 How It Works

- User submits a startup URL
- SpyGlass AI crawls the website
- AI agents extract and analyze business information
- Competitor discovery agents search the web and Product Hunt
- SEO and positioning analysis is generated
- SWOT and strategic insights are created
- - Results are stored in MongoDB
- Users access insights through a dashboard and AI chat assistant

# 🛠️ Tech Stack

- Frontend: Next.js, TailwindCSS
- Backend: Next.js API Routes
- Database: MongoDB
- AI Models: Gemini API
- Web Crawling: Anakin Crawl API
- Authentication: JWT
- Language: TypeScript

## 📦 Installation

- Clone the repository:

```bash
git clone https://github.com/GauravKesh/spyglass-ai
cd spyglass-ai
```

- Install dependencies:

- npm install

### ⚙️ Environment Variables

- Create a .env.local file:

```bash
MONGODB_URI=

JWT_SECRET=

GEMINI_API_KEY=

ANAKIN_API_KEY=

PRODUCT_HUNT_ACCESS_TOKEN=
```

### ▶️ Run Development Server

```bash
npm run dev
```

Open:

http://localhost:3000
